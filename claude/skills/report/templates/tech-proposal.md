# Technical proposal template

## 使用说明

技术提案是**请求批准做一件大事**的文档。读者要据此判断：这事该做吗 → 该用这个方案吗 → 投入值得吗 → 风险可接受吗。提案的目的不是展示你思考多深，而是让读者用最少时间做出有把握的批准/拒绝决定。

**适用场景**：RFC、架构提案、新模块/系统设计、大型重构方案、新依赖引入
**不适用**：日常开发任务（直接 /design）、多方案权衡（用 decision）
